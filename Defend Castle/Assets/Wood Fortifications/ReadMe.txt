Create your own Fortifications around your town with the Wood Fortifications pack.
The models can be used on Mobil or any other platform, it contains a Low Poly version of the structures.
It also has Hand painted High Quality textures.
You can create several different designs with the wall, towers, gate, barricades and ladders as sown on the Screenshots.
It uses a Texture for the walls - 2048x2048 TGA
and another for the Hay - 1024x1024 TGA

The Pack is grouped in a main folder named "Wood Fortifications " and it has a Demo Scene, shown on the Screenshots, and the Prefabs folder, which contains all the elements, grouped in Barricades, Ladders and Walls.
